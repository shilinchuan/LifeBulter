# LifeButler 变更日志

## 2026-06-03

### 二期基础增强

- 新增四象限今日任务清单，任务通过 `today_date` 和 `quadrant` 进入今日视图。
- 未完成今日任务支持自动顺延显示，已完成任务不再显示在今日四象限。
- 新增番茄钟，支持绑定今日任务、开始、暂停、继续、结束，并记录专注 session。
- 新增 `pomodoro_sessions` 表，记录任务、开始时间、结束时间、计划分钟、实际分钟、状态和备注。
- 新增 schema 迁移机制，数据库目标版本升级到 version 3。
- 迁移前自动备份旧数据库到 `data/lifebutler_backup_before_migration_*.db`。
- 兼容旧预算表迁移；预算功能后续已从界面移除。
- 重做整体深色主题，并支持导航栏切换浅色/深色。
- 图表改为仪表盘风格，支持 donut 饼图、柔和折线图和柱状图。
- 健康模块补充体重、运动记录的编辑和删除入口。
- 增加轻量 smoke tests，覆盖迁移、今日任务、番茄钟、主窗口和图表。

### UI 修复与可读性迭代

- 修复 PyQt6 托盘图标枚举错误，避免应用启动崩溃。
- 修复托盘图标不显示、窗口隐藏后难以恢复的问题。
- 修复新增记录等弹窗中下拉框、日期框、金额框显示不完整的问题。
- 为下拉框、日期框、数字步进框增加 SVG 箭头资源，解决右侧箭头不可见。
- 调整全局表单控件的最小高度、最小宽度、右侧按钮空间和下拉列表样式。
- 扩大记账、待办、健康、备忘录弹窗尺寸和表单行距。
- 修复浅色模式下图表仍保持深色的问题，`ChartWidget` 增加 `set_theme()` 并随全局主题重绘。
- 修复四象限今日任务和任务池同步不及时的问题，切换标签、编辑象限、完成任务、删除任务后统一刷新。
- 任务池工具条拆成两行，避免“加入今日”和筛选控件拥挤。
- 四象限页面调整为左侧四象限概览、右侧番茄钟布局。
- 四象限小卡片从多列表格改为概览列表，仅显示任务和专注概览，避免标题被截断。
- 每个象限增加“放大查看”按钮，双击象限任务也可打开该象限大屏列表。
- 象限大屏列表显示任务、截止日期、番茄数和累计分钟。
- 番茄钟按钮多次微调，最终布局为第一排 `开始 / 暂停 / 结束`，第二排 `完成任务`。
- 将原“停止”按钮文案调整为“结束”。

- 统一月度统计图表首次绘制和按钮切换的状态来源，减少首屏与切换后的视觉跳变。
- 统一四个模块操作按钮的 emoji 文案，新增类按钮使用主题主色样式适配深浅模式。
  - 主窗口构造、图表深浅主题切换。
  - 任务池与四象限同步、象限大屏弹窗构造。
