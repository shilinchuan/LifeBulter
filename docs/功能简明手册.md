# LifeButler 用户测试简明手册

LifeButler 是一个本地运行的生活管理桌面应用，用来管理记账、目标、待办、今日四象限、番茄钟、健康记录、备忘录和周报复盘。数据默认保存在本机 `data/lifebutler.db`，不会上传到服务器。

## 运行前准备

请先确认本机已安装 Python 3.10 或更高版本。

在项目文件夹中打开终端，执行：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Windows PowerShell 可参考：

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## 启动方式

macOS 可以双击：

```text
启动 LifeButler.command
```

也可以在终端运行：

```bash
.venv/bin/python main.py
```

窗口左上角关闭按钮会把应用隐藏到托盘；真正退出请在启动应用的终端按 `Ctrl+C`，或使用托盘菜单里的退出。

## 建议测试路径

1. 首页驾驶舱
   - 试试快速收集，例如 `午饭 28r`、`跑步 30 分钟`、`明天提交报告`。
   - 看首页的今日任务、财务、健康和生活雷达是否容易理解。

2. 记账管理
   - 新增一条支出和一条收入。
   - 查看月度统计和图表。
   - 尝试编辑、删除记录。

3. 目标管理
   - 新增一个 Objective。
   - 新增 KR，并更新 KR 进度。
   - 新增 Project。Project 只属于目标管理，不绑定待办任务。
   - 双击目标、KR、Project 行查看详情，再从详情里编辑或删除。

4. 待办事项
   - 在任务池新增任务。
   - 双击任务行应进入编辑，不应直接切换完成状态。
   - 用“完成/恢复”按钮切换任务状态。
   - 把任务加入今日四象限，切到“今日四象限”查看。

5. 番茄钟
   - 在今日四象限选择一个任务。
   - 点击开始、暂停、结束。
   - 完成一次专注后观察番茄/分钟统计。

6. 健康记录
   - 添加体重记录。
   - 添加运动记录。
   - 查看趋势图和统计是否清楚。

7. 备忘录
   - 新增一条备忘录。
   - 尝试分类、搜索、置顶和预览。

8. 周计划 / 周报复盘
   - 从任务池加入本周。
   - 设置完成度和进展备注。
   - 标记为今日或取消今日。
   - 填写三句复盘并导出 Markdown。

9. 设置
   - 查看数据库路径是否完整可读。
   - 尝试备份数据库。

## 反馈清单

测试时请重点记录：

- 哪个页面或按钮看不懂。
- 哪些文字被截断、重叠或显示不清楚。
- 哪些地方点击后没有反应，或反应和预期不同。
- 是否能独立完成“记一笔账、建一个目标、建一个任务、做一次周报”。
- 深色/浅色模式下是否都能正常阅读。
- 你希望删除、合并或简化的功能。

## 注意事项

- 这是源码测试包，不是正式安装包。
- 压缩包不包含 Python 环境，需要按上面的步骤安装依赖。
- 第一次运行会自动创建本机数据库。
- 用户测试数据只保存在你的电脑上。
